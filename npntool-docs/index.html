npntool-0.1.0.1: Modelling and analysis toolset for nested Petri nets

Nested Petri nets is an extension of Petri net formalism with net tokens for modelling multi-agent distributed systems with complex structure. While having a number of interesting properties, NP-nets have been lacking tool support. NPNTool presents a way to define and model (nested) Petri nets in a modular way in addition to a number of algorithms for analysing them.
